# yama-cli Documentation

This folder is for **CLI-specific** documentation: command references, usage
guides, and recipes specific to the `yama` command-line tool.

## Format specifications

Format specifications (YAMAML, SimpleDSP, OWL-DSP, including verbatim Japanese
originals) live on the documentation site. The CLI follows those specs — it
does not define them.

- **YAMAML Specification** — <https://docs.yamaml.org/specs/yamaml/spec/>
- **SimpleDSP Specification** — <https://docs.yamaml.org/specs/simpledsp/spec/>
- **OWL-DSP Specification** — <https://docs.yamaml.org/specs/owl-dsp/spec/>
- **PKL Authoring Guide** — <https://docs.yamaml.org/specs/yamaml/pkl/>
- **Worked Examples** — <https://docs.yamaml.org/specs/yamaml/examples/> and <https://docs.yamaml.org/specs/simpledsp/examples/>

## Contributing CLI docs

If you are adding documentation about a `yama` command or CLI workflow, add it
here. Content about the underlying formats should be contributed to the
[docs site repository](https://github.com/yamaml/docs) instead.
